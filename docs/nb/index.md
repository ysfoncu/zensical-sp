# MOSO hjelp og onboarding-senter

Velkommen! Dette nettstedet leder deg gjennom **MOSO-testscenarioene** steg for steg,
med skjermbilder for hvert trinn.

## Slik fungerer det

1. **Skriv inn e-postadressen din** — den brukes kun til å følge fremdriften din og
   samle inn tilbakemeldingene dine. Ingen passord kreves.
2. **Arbeid deg gjennom scenarioene** i sidemenyen, og følg trinnene i den
   faktiske applikasjonen mens du leser.
3. **Send inn tilbakemelding nederst i hvert scenario** — fortell oss om det var
   nyttig, og legg gjerne igjen en kommentar. Når du sender inn tilbakemelding,
   markeres scenarioet som fullført.
4. Sjekk **Min fremdrift** når som helst for å se hva du har fullført.
