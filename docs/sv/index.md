# MOSO hjälp- och onboardingcenter

Välkommen! Den här webbplatsen guidar dig genom **MOSO-testscenarierna** steg för steg,
med skärmbilder för varje moment.

## Så fungerar det

1. **Ange din e-postadress** — den används endast för att följa dina framsteg och samla in
   din feedback. Inget lösenord behövs.
2. **Arbeta dig igenom scenarierna** i sidofältet och följ stegen i den
   riktiga applikationen medan du läser.
3. **Skicka in feedback längst ner i varje scenario** — berätta om det var
   användbart och lämna gärna en kommentar. När du skickar in feedback markeras
   scenariot som slutfört.
4. Öppna **Mina framsteg** när som helst för att se vad du har slutfört.
